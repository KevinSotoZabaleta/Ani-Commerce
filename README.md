# Ani-Commerce

## Autor

**Kevin Soto Zabaleta**

## Descripción

Proyecto sobre ventas de tarjetas de anime

- One Piece
- Dragon Ball
- Kimetsu no yaiba

## Instalación

Instalación para hacer uso de spa

1.- **Clonar Repositorio**

https://github.com/KevinSotoZabaleta/Ani-Commerce

cd Ani-commerce

2.- **Instalar Dependencias**

- npm i

3.- **Ejecutar Proyecto**

- npm run dev 

## Dependencias

- React-router-dom
- Firebase
